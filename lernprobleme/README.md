# Lernprobleme und Lösungen · Deutsch A2.2

Interaktive Präsentation mit genau neun Folien für Attia und Abdullah.

## Funktionen

- schrittweise Animationen innerhalb der Folien
- Tastatursteuerung mit Pfeiltasten und Leertaste
- Wischgesten auf Mobilgeräten
- deutsches Vorlesen pro Folie oder für das ganze Skript
- vollständiges Sprecher-Skript im seitlichen Bereich
- Vollbildmodus und PDF-Druck
- responsive Darstellung im Format 16:9

## Start

Die Datei `index.html` direkt öffnen oder einen lokalen Webserver starten:

```bash
python -m http.server 8000
```

Danach `http://localhost:8000` öffnen.
